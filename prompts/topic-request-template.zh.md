# 主题请求模板

当你把仓库链接发给别人的 OpenClaw 后，可以用这个简短模板指定具体讲义任务。

```text
请按这个 repo 的 OpenClaw handoff 规范生成中文数学讲义：
https://github.com/pkumath/chinese-math-lecture-notes-repro

主题：
- <写主题，例如：Trotter--Kato theorem for Markov semigroups>

源材料：
- <论文/书/网页/本地文件/你贴的草稿>

必须覆盖的 source/request spine：
- <对象 1>
- <对象 2>
- <对象 3>

输出目标：
- Markdown / Notion / both

额外要求：
- <例如：要有完整证明；重点讲 Gaussian conditioning；不要讲实现 API；面向第一次读 Tensor Programs 的读者。>

请先读 repo 的 `OPENCLAW_HANDOFF.md`，然后本地起草、运行 validator、做证明颗粒度 audit。完成时给出文件路径/链接、validator 结果、spine 覆盖和 audit 结论。
```
